Trabalho de ED II - Estrutura de Dados II

O objetivo do trabalho é desenvolver um site de e-commerse em 3 etapas, incluindo os conceitos de organização de arquivos, como uma aplicação Web, utilizando o framework Flask

O trabalho divido em 3 etapas, onde na 1º etapa deve ser criado um tipo de catálogo online, onde aqueles que entrarem na aplicação WEB poderão ver diversos produtos disponíveis, fazer consultas desses produtos e compras. Os produtos virão de um dicionário feito em python.

Na 2ª etapa o objetivo será criar arquivos indexados utilizando árvores do tipo B para registrar as vendas feitas no site

Na 3ª etapa, utilizando dos conhecimentos de grafos deve ser criado um sistema de entrega de produtos, que seja capaz de calcular as rotas, informando ao cliente o preço do frete e tempo estimado de entrega
